This folder contains media assets for the Love Portal website.

- heart.png: Heart icon (use for custom icons, backgrounds, or gallery)
- heart-1.png: Alternate heart icon (use for variety in UI or gallery)
- audio.mp3: Custom background music (use as the main audio in index.html and love.html)

To use these assets:
- For images: <img src="media/heart.png" ...>
- For audio: <audio ...><source src="media/audio.mp3" type="audio/mpeg"></audio>

You can reference these files in your HTML for a more personalized and branded experience.
